# NoProfit4Us Application

Questo è un progetto web per la gestione di un'associazione no-profit (APS), composto da:
- **Frontend**: Vue.js 3 + Vite + Javascript (con PrimeVue).
- **Backend**: Python + FastAPI + SQLAlchemy + SQLite.

## Struttura del Progetto

- **[frontend/aps-frontend](file:///C:/Users/a264666/OneDrive%20-%20Enel%20Spa/Source/StudioPersonale/noprofit4us/application/frontend/aps-frontend)**: Applicazione client realizzata in Vue 3 e Vite.
- **[backend](file:///C:/Users/a264666/OneDrive%20-%20Enel%20Spa/Source/StudioPersonale/noprofit4us/application/backend)**: API server realizzata in Python e FastAPI.

## Prerequisiti

### Frontend
- Node.js (v18+)
- npm / yarn / pnpm

### Backend
- Python (v3.10+)
- Virtualenv (opzionale ma consigliato)
